# Spaceship Skill

Respect any constraints on sci-fi scope that are active in this session. If the scope has been narrowed — by conversation, context, or instruction — apply that constraint to all 5 entries. Otherwise, draw freely from across sci-fi literature.

Generate a list of 5 spaceships from books. All spaceships must come from novels or other written works — not films, TV shows, or games (unless the game/show originated from a book). For each entry include the spaceship name, the book it appears in, and the author. Present the list cleanly. Do not add commentary beyond the list unless the user asks a follow-up question.
